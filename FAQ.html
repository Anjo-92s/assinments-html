Frequently Asked Questions

Question 1: What is HTML?
Answer :HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages and web applications.

Question 2: What are the basic tags in HTML?
Answer :Some of the basic tags in HTML include <html>, <head>, <body>, <p>, <h1> to <h6>,
<a>, <img>, <ul>, <ol>, <li>, <table>, <tr>, <td>, etc.

Question 3: What is the purpose of CSS?
Answer :CSS (Cascading Style Sheets) is used to style and format the layout of a document written in HTML or XML.
It allows you to control the appearance of your web pages.

Question 4: What is JavaScript?
Answer :JavaScript is a programming language that allows you to add interactivity and dynamic features to your web pages.
It is commonly used for client-side scripting.

Question 5: What is the difference between <span> and <div> tags?
Answer :The <span> tag is an inline element used for styling a specific part of the text or an element, while
the <div> tag is a block-level element used for grouping and formatting larger sections of content.
